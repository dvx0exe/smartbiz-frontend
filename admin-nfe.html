Habilitar Módulo NF-e
Preencha as credenciais da Focus NFe / Enotas para este negócio. Após salvar, o módulo NF-e fica desbloqueado automaticamente no painel do cliente.
Use homologação para testar. Mude para produção quando o cliente estiver com tudo ok.
Focus NFe: focusnfe.com.br → Configurações → Token
Desabilitar NF-e
⚠️ Ao desabilitar, o módulo NF-e fica bloqueado imediatamente para este negócio. O cliente verá a tela de "módulo não habilitado" na próxima vez que acessar.

As credenciais serão removidas. Para reativar, será necessário cadastrá-las novamente.
